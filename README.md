# TeamworkImages
Arquitectura de Computadores, trabajo final. (21-22)

## Resumen
Trabajo en grupo desarrollado en dos fases:
- [ ]  Primera fase: crear algoritmo que operere sobre componentes RGB de una imagen de una manera determinada.
- [ ]  Segunda fase: desarrollar el código anterior de modo que funcione en SIMD y multihilo.

## Software utilizado
Ubuntu 20.04 LTS en Oracle VirtualBox, Visual Studio Code, git, bash,...
